# Portfolio

A modern, responsive portfolio website showcasing my frontend development skills, projects, and background.

Live site:
https://joakol-portfolio.netlify.app/

---

## Description

This is my personal portfolio website, designed to present my work, technical skills, and background in a clean, structured and user-friendly way.

The latest version focuses on clarity, simplicity and strong project presentation — making it easy for recruiters and developers to quickly understand what I’ve built and what I can do.

### Features

- Clean and modern hero section with introduction
- Featured projects with detailed descriptions and tech stack
- Responsive layout across mobile, tablet and desktop
- Clear call-to-action buttons for contact and navigation
- Structured project cards with highlights and live links
- Minimal and professional UI

---

## Built With

This project is built using:

- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Font Awesome](https://fontawesome.com/) (icons)
- [Netlify](https://www.netlify.com/) (deployment)

> Note: This version is intentionally built without JavaScript to keep the experience fast, simple and focused.
